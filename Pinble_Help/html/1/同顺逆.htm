数字三『彩神通』常用术语
同顺逆
     
 

在三维元素（012路、大中小、升平降、邻孤传、复隔中、重斜跳、热温冷等）的单选指标中，把该指标的三个元素定义成有一定顺序的循环圈；如果两期号码在同一位置上的状态相同则称为同，在同一位置上的状态变化与定义的顺序一致则称为顺，在同一位置上的状态变化与定义的顺序相反则称为逆。同顺逆指标是反映两期号码之间三维元素的单选状态转换的一个指标。
　　012路的顺序为：0路 → 1路 → 2路 → 0路……；
　　大中小的顺序为：大 → 中 → 小 → 大……；
　　升平降的顺序为：升 → 平 → 降 → 升……；
　　邻孤传的顺序为：邻 → 孤 → 传 → 邻……；
　　复隔中的顺序为：复 → 隔 → 中 → 复……；
　　重斜跳的顺序为：重 → 斜 → 跳 → 重……；
　　热温冷的顺序为：热 → 温 → 冷 → 热……；

　　例如：上期号码为129（单选120路、小小大），本期号码为387（单选021路、中大大）
　　　　　则012路的同顺逆状态是逆同顺；
　　　　　则大中小的同顺逆状态是逆顺同。
