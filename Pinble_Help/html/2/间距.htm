数字三『彩神通』常用术语
间距
     
 
数字三型彩票的间距为第一间距、第二间距、第三间距中的最大值，间距分单选间距和组选间距。
 
     
 
第一间距是首间距和尾间距的总和，即：第一间距 = 首间距 + 尾间距。
 
     
 
首间距是指号码中第一位数字和数字0的差值，即：首间距 = 号码中第一位数字 - 0；如：号码283的第一位数字是2，它的首间距则为2 - 0 = 2。
 
     
 
尾间距是指数字9和号码中第三位数字的差值，即：尾间距 = 9 - 号码中第三位数字；如：号码283的第三位数字是3，它的尾间距则为9 -3 = 6。号码283的第一间距等于首间距2加上尾间距6等于8。
 
     
 
第二间距是第一位数字和第二位数字的差值(绝对值)，如：号码283的第一位数字是2，第二位数字是8，号码283的第二间距则为 8 - 2 = 6。
 
     
 
第三间距是第二位数字和第三位数字的差值(绝对值)。如：号码283的第二位数字是8，第三位数字是3，第三间距则为 8 - 3 = 5。
 
     
 
由于号码的间距是第一间距8、第二间距6和第三间距5中的最大间距值。所以号码283的间距为8。
 
     
 
一个号码可以有单选间距和组选间距。上面的号码283就是一个单选号码，它的间距8也就是单选间距。
 
     
 
号码283的组选是238，那它的首间距为2 - 0 = 2，尾间距为9 - 8 = 1，第一间距为2 + 1 = 3；第二间距为3 - 2 = 1；第三间距为8 - 3 = 5；所以组选号码238的间距是5(1、3、5里面的最大值)。从这里可以看出来一个号码的单选间距和组选间距不一定是一样的。
