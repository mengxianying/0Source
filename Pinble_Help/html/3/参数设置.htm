『拼搏在线』系列彩票软件—排列五『彩神通』

参数设置

     
 
参数设置中可以在这里修改当前设置，用鼠标双击某一项就会弹出对话框，用户可以对“当前彩票”、“当前玩法”、“分析方式”、“数据范围”、“数据间隔”、“排列方式”、“奖号筛选”、“数据来源”、“热温冷期数”进行设置。
  
 
 
     
 

分析方式：双击“当前玩法”即可显示下面的设置界面。

分析方式：双击“分析方式”即可显示下面的设置界面。

 

　　在该窗口中系统缺省列出了“原始数据”、“玩法123位”、“玩法234位”、“玩法345位”等，用户可以自己选择系统设置的方式，也可以自己任意添加其它位置成为新的分析方式，还可以修改已经设置的名称、位置并能改变显示的顺序。而且每种分析方式都有快捷键来进行快速切换。快捷键具体定义如下：
　　“原始数据”　　　　 Ctrl + ~
　　“玩法123位”　　　　Ctrl + 1
　　“玩法234位”　　　　Ctrl + 2
　　“玩法345位”　　　　Ctrl + 3
　　“玩法12位”　　　　 Ctrl + 4
　　“玩法23位”　　　　 Ctrl + 5
　　“玩法34位”　　　　 Ctrl + 6
　　“玩法45位”　　　　 Ctrl + 7
　　比如选择分析方式为“玩法123位”后，系统自动显示排列五开奖号码的第1位、第2位、第3位对应的走势图表和缩水条件，也就是说此时的分析图表和选号软件都是按“数字三『彩神通』专业版”的功能进行展开的，所有的软件功能、操作方法、思路都与“数字三『彩神通』专业版”没有任何区别，只是显示的数据是排列五开奖号码的第1位、第2位、第3位的数字。所有的选号缩水条件也都只对排列五开奖号码的第1位、第2位、第3位的数字进行操作，与其他数据没有任何关系。无论怎么变换“分析方式”，缩水器中的排列五号码都不会改变，这样我们就可以通过不同的“分析方式”继续对缩水器中的排列五号码进行缩水，达到最理想的投注数量，从而可以看见别人看不见的线索，抓住别人抓不到的机会，从另一个角度来轻松博取10万大奖。

　　分析方式中有关“数字三『彩神通』专业版”的操作请查看“数字三『彩神通』专业版”的操作帮助说明书，在这里就不再仔细说明。

注：关于玩法与分析方式的区别请参看玩法与分析方式的区别及操作使用演示录像

  
     
 

数据范围：双击“数据范围”即可显示下面的设置界面。
用户可以在这里选择数据的范围，比如“所有号码”、“最近一年的号码”、“起始期号与终止期号”、“快速选择”等等。系统缺省选择为“所有号码”。

  
 

数据间隔：双击“数据间隔”即可显示下面的设置界面。
用户可以在这里选择数据的间隔，比如“无间隔”、“间隔N期”、“期号尾数”、“期号余数”、“每周的星期”、“每月的日期”等等。系统缺省选择为“无间隔”。
当按间隔期数方式显示时，可以按快捷键Ctrl + “加号”来增加间隔期数，按快捷键Ctrl + “减号”来减少间隔期数，按快捷键Ctrl + 0来恢复成“无间隔”显示。

  
     
 

排列方式：双击“排列方式”即可显示下面的设置界面。
用户可以在这里选择开奖号码显示的排列方式，比如按“单选排列方式”显示、按“组选排列方式”显示。系统缺省选择为“单选排列方式”显示。

  
     
 

奖号筛选：双击“奖号筛选”即可显示下面的设置界面。
用户可以在这里选择自己的条件来筛选开奖数据，比如只显示组三或者组六的开奖号码。

  
     
 

数据来源：双击“数据来源”即可显示下面的设置界面。
用户可以在这里选择不同数据来源当成开奖数据来显示，比如按开奖号码的距离来显示，此时图表中显示的开奖号码是开奖号码的距离，所有的图表都是开奖号码距离的图表。系统缺省选择为“开奖号码”。

  
     
 

热温冷期数：双击“热温冷期数”即可显示下面的设置界面。
用户可以在这里设置热温冷期数，系统会根据用户设置的热温冷期数自动设定热码、温码和冷码。系统缺省的热温冷期数为7期。
